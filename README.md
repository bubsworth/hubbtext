## Running the Application

To run this application there are a few requirements:

1. Node v20
2. Yarn
3. Docker

Once cloned, run the following to get the app started

1. `yarn install`
2. `yarn start`
3. Open a web browser of choice
4. Go to `http://localhost:3000`
